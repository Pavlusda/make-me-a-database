Make me a Database!

Use this page to practice creating a database (using MongoDB Atlas) and interacting with data values by modifying the code. The point of this database is to store a list of potential pet names. Enjoy!

Follow these steps to run your server:

  1. Click on the Windows icon and search for "command prompt" to open your terminal
  2. Navigate to this project directory ending in ".../practice"
  3. Once inside the project directory type the command "node server" and press enter
  4. You should see a confirmation message saying that the server is running. You can now open a browser and type "localhost:3000" into the URL to open the application
  5. To close the server press CTRL + C in the command prompt, save your changes in Visual Studio Code, and restart the server to view changes
